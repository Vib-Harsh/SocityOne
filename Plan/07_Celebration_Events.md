# Celebration & Common Event Section

## Overview
A community engagement tool for managing society-wide events, festivals, and gatherings.

## Key Features
- **Event Notices**: Admins can broadcast details about upcoming celebrations (e.g., Diwali party, Independence Day flag hoisting).
- **RSVP System**: Residents can confirm their attendance and specify the number of family members joining.
- **Event Gallery**: A shared photo gallery for residents and admins to upload pictures after an event.
- **Polls/Surveys**: Gathering resident opinions for event planning (e.g., choosing catering menus).
