# Document Storage Locker

## Overview
A secure vault for both the society administration and individual residents to store sensitive documents.

## Key Features
- **Encrypted Storage**: Documents must be stored in an encrypted format to ensure privacy and security.
- **Society Documents**: Admins can store society registration certificates, audit reports, fire safety certificates, etc.
- **Resident Documents**: Residents can store identity proofs (Aadhar, PAN), rent agreements, vehicle RCs, and property deeds.
- **Access Control & Sharing**: Granular controls allowing residents to temporarily share specific documents with the Admin (e.g., submitting a lease agreement).
