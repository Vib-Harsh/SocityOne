# Stay Module

## Overview
The Stay Module is responsible for managing the physical properties within the society.

## Key Features
- **Property Types**: Categorizing properties into Flats, Rooms, Bungalows, Penthouses, etc.
- **Property Details**: Storing dimensions, block/tower numbers, floor numbers, and specific unit numbers.
- **Property Status**: Tracking whether a property is currently occupied, vacant, under construction, or locked.
- **Amenities Mapping**: Linking specific properties to dedicated parking spots or special amenities.
