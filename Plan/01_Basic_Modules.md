# Basic Modules

## Overview
This module handles the core foundation of SocityOne, dealing with authentication, authorization, and access control.

## Key Features
- **User Management**: Registration, login, profile management, and password recovery.
- **Role Management**: Defining roles (e.g., Super Admin, Society Admin, Owner, Tenant, Guard, Staff).
- **Permission Management**: Granular access control defining what each role can view or modify.
- **Subscriptions**: Managing the SaaS aspect if SocityOne is sold to multiple societies. Tracks subscription plans, limits, and renewals per society.
